Экзаменационные вопросы  (ООП. Java)

1.      Основные принципы языка Java: объектно-ориентированный подход, полиморфизм, ссылочные типы, интерпретация. Сравнение с Си++.

2.      Базовые типы данных. Класс String, его особенности и методы. Классы – обертки для базовых типов данных (int и Integer), их назначение.

3.      Базовые типы данных и ссылочные типы. Особенности работы со ссылочными типами, динамическое распределение памяти, «сбор мусора». Операции над ссылками.

4.      Наследование и приведение типов. Класс Object. Расширение и сужение ссылочного типа при переходе от класса к классу. Операция instanceof. Встроенный полиморфизм методов.

5.      Объектная модель JVM. Классы Object и Class, их методы. Организация JVM. Система мета-объектов – описателей классов. Динамическая загрузка классов в JVM.

6.      Статические элементы классов. Их использование для задания констант и функций. Метод main.

7.      Модульная структура Java-программы. Пакет, класс, метод. Полные и сокращенные имена классов и методов. Объявление и подключение (импорт) пакетов.

8.      Области видимости: проект, пакет, класс, метод, производный класс. Модификаторы доступа (public, private, protected) и модификаторы изменения (static, final).

9.      Инициализаторы и конструкторы. Многообразие вариантов инициализации объектов. Явный вызов конструкторов

10.  Массивы в Java. Массив как ссылочный тип и как объект. Реализация объекта-массива как динамического массив указателей (ссылок). Размерность массива, инициализация массивов. Многомерные массивы. Приведение типа массива.

11.  Наследование в Java. Древовидная система классов при одиночном наследовании. Класс Object, его назначение. Приведение типов ссылок при работе с системой наследуемых классов. Работа с данными базового класса. Доступ к базовому классу (super).

12.  Встроенный полиморфизм методов в Java. Интерфейс (interface). Использование интерфейсов для доступа к объектам различных классов. Наследование интерфейсов. Абстрактный базовый класс.

13.  Исключения, их генерация и обработка. Классы исключений.

14.  Копирование объектов. Интерфейс Clonable.

15.  Классы структур данных. Представление список и деревьев в Java.

16.  Классы оконных приложений (Frame и Applet). Программирование апплетов. Работа с графикой (классы Graphics, Color, Font).

17.  Классы элементов управления (Component, Canvas, Label, Button, TextField, TextArea, Chioce, List, MenuBar, Menu, MenuItem). Их основные методы. Конструирование объектов управления (списков, меню, в том числе иерархических), их объявление в классе окна.

18.  Обработка событий в Java. Классы событий, интерфейсы слушателей, объявление слушателей событий. Методы обработки событий (примеры).

19.  Потоки и их программирование.

20.  Синхронизация потоков. Блокирование потоков, пример: модель поставщик-потребитель.

21.  Понятие протокола. 7-уровневая модель сетевых протоколов, назначение уровней. Виды сетевого сервиса: логический канал, клиент-сервер.

22.  Пакет Java.net. Установление соединений через сокеты. Использование потоков при программировании соединений.

23.  Программирование ввода-вывода. Классы потоков ввода-вывода. Ввод-вывод текстовых файлов.

24.  Двоичные файлы и их форматы. Двоичные потоки. Объектовые потоки. Сериализация.
