Контрольная работа. Массивы указателей, списки и иерархические СД

1.      Функция получает на вход строку и возвращает динамический массив указателей на слова строки. Для каждого слова создается копия в динамическом массиве, на который помещается ссылка в массив указателей.

2.      Функция получает на вход строку и возвращает односвязный список, каждый элемент которого содержит указатель на копию слова строки в динамической памяти.

3.      Функция получает массив целых и возвращает массив указателей на положительные элементы исходного массива, упорядоченные по возрастанию (сортируется массив указателей, исходный массив не меняется).

4.      Функция получает строку и подстроку и возвращает динамический массив указателей на все вхождения подстроки в строку (с использованием NULL-ограничителя).

5.      Элемент односвязного списка содержит N указателей на строки. Функция читает файл и заполняет структуру данных указателями на прочитанные строки (список и строки размещаются в динамической памяти).

6.      Двухуровневый массив указателей на строки. Статический массив верхнего уровня содержит N указателей на динамические массивы указателей нижнего уровня (размерностью N). Функция читает файл и заполняет структуру данных указателями на прочитанные строки.

7.      Статический массив указателей содержит N заголовков односвязных списков (указателей). Элемент списка содержит указатель на строку. Функция читает файл и заполняет структуру данных. Размерность каждого списка ограничивается N элементами.

8.      Элемент односвязного списка содержит заголовок односвязного списка (Двухуровневый список). Элемент списка второго уровня содержит указатель на строку. Функция читает файл и заполняет структуру данных. Размерность каждого списка ограничивается N элементами.

9.      Функция получает односвязный список и «нарезает его на части» по N элементов, возвращая динамический массив указателей на полученные списки.

10.  Функция получает односвязный список и «нарезает его на части» по N элементов, создает и возвращает список, каждый элементы которого содержат заголовки полученных списков.

11.  Сортировка односвязного списка вставками.

12.  Сортировка односвязного списка выбором.

13.  Сортировка двусвязного списка вставками.

14.  Сортировка двусвязного списка выбором.

15.  Сортировка циклического списка выбором

16.  Сортировка циклического списка вставками.

17.  Элемент двусвязного циклического списка содержит указатель на строку. Строки упорядочены по возрастанию. Вставить строку в список с сохранением упорядоченности. В список помещается копия входной строки в динамической памяти.

18.  Элемент односвязного списка содержит массив указателей на строки. Функция возвращает указатель на строку максимальной длины.

19.  Сортировка рекурсивным разделением. Функция получает массив размерности N и вычисляет медиану – среднее арифметическое, создает два динамических массива размерности N и копирует туда элементы меньшие и большие медианы. Затем вызывает рекурсивно сортировку частей и «сливает» упорядоченные части в исходный массив.

20.  Имеется функция char *read_word(FILE *fd), которая читает из файла очередное слово, символ конца строки (возвращает пустую строку) или конец файла (возвращает NULL). С ее помощью заполнить структуру данных: элемент динамического массива содержит слово, число его повторений и заголовок списка, элемент списка содержит номер строки, где оно встречается.

21.  Имеется функция char *read_word(FILE *fd), которая читает из файла очередное слово, символ конца строки (возвращает пустую строку) или конец файла (возвращает NULL). С ее помощью заполнить структуру данных: элемент списка содержит слово, число его повторений и динамический массив с номерами строк, где оно встречается.
