Лаб.раб.№1. Изучение среды программирования NetBeans. Разработка простого оконного приложения.

Задание (10 баллов).

  1. Разработать класс для массива целых чисел (конструктор для пустого массива и массива, заполненного случайными числами, методы добавления, удаления, вывода содержимого в строку).
  2. Для  варианта задания из 4.2. (Материал в главе 3 «Проектирование программ») добавить метод, реализующий требуемый алгоритм.
  3. Разработать консольное приложение (main), тестирующее указанный вариант задания.
  4. Разработать простое оконное приложение, в котором предусмотреть текстовое поле для ввода данных в массив, поле TextArea для вывода результатов, кнопки (пункты меню) для добавления, удаления и просмотра данных. Все манипуляции производить с объектом класса, разработанного в пп.1,2.

 

Защита (4 балла). Создать в NetBeans пустое консольное приложение с main. Используя программные заготовки из 2.5 (2_Вопросы без ответов), «оживить» по 2 фрагмента, задав для них статические входные данные. Результат вывести с помощью System.out.println в окно отладки. Провести анализ фрагментов, выделить и закомментировать стандартные программные решения (контексты) содержательно сформулировать результат их работы. (Материал в главе 2 «Анализ программ»).

 

Лаб.раб.№2. Работа с графикой  и программирование потоков.

Задание (14 баллов)

  1. Разработать абстрактный класс движущегося (изменяющегося) графического объекта. Данные класса – координаты центральной точки, размеры, цвет. Варианты движения и соответствующие данные класса (для разных бригад): равномерное движение с отражением от границ панели, вращение, пульсация (одновременное изменение размеров, изменение размеров в противофазе, изменение яркости, цвета по одной или нескольким компонентам).  Разработать интерфейс (абстрактные методы) управления объектом - прорисовка, стирание, проверка на принадлежность объекту точки  с заданными координатами (для выбора) объекта, изменения параметров объекта за 1 шаг движения.
  2. Разработать 2 производных класса изменяющихся объектов одного типа (например, вращающийся многоугольник и смайлик, или движущаяся строка текста и квадрат). Разработать класс потока, управляющего движением (изменением) одного объекта и протестировать его работу.
  3. Разработать класса потока, управляющий движением группы графических объектов, заданных вектором ссылок. Разработать оконное приложением, в котором реализовать добавление и удаление объектов кликом мыши, а также остановку и возобновление параллельного движения объектов.

 

Защита (4 балла).. Разобрать (содержательно проанализировать и закомментировать) по 2 теста из тестовых заданий (номера тестов выдаются преподавателем)

 

Лаб.раб.№3 (10 баллов). Работа с файлами. Добавить в оконное приложение для предыдущей работы средства сохранения и загрузки движущейся картинки в текстовый, двоичный и сериализованный файл (все параметры объектов) – меню и диалоги выбора файла, а также методы, обеспечивающие требуемый формат хранения объектов.

 

Защита (4 балла). Разобрать (содержательно проанализировать и закомментировать) по 2 теста из тестовых заданий (номера тестов выдаются преподавателем)

 

Лаб.раб.№4. (10 баллов) Работа с сетью. Добавить в оконное приложение для предыдущей работы «слушатель» - серверную компоненту, реализованную в виде отдельного потока.  Задается номер порта, на котором находится слушатель. При выборе мышью движущегося объекта происходит установления соединения со «слушателем» аналогичного приложения, после чего создается поток, через который передаются данные объекта (формат потока можно взять из предыдущей работы). Слушатель, получив данные объекта, создает его и добавляет в собственный вектор. В программе предусмотреть список адресов (IP-порт), на который можно рассылать объекты.

 

Защита (4 балла). Разобрать (содержательно проанализировать и закомментировать) по 2 теста из тестовых заданий (номера тестов выдаются преподавателем).
